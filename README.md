

# Ethereum Price Fetcher (Python)

This is a simple Python script that fetches the current Ethereum (ETH) price in USD using the free CoinGecko API and saves the result as a JSON file.

## 🚀 Features
- Uses a **free public API** (no API key required)
- Fetches real-time Ethereum price in USD
- Saves response to a local JSON file
- Minimal and easy-to-understand code

## 📦 Requirements
- Python 3.x
- `requests` library

Install dependencies:
```bash
pip install requests
